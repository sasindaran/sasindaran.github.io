Timetable Pro

Efficient and automated academic scheduling system with real-time conflict detection.

Project Overview

This project introduces a dynamic scheduling system for academic institutions, featuring advanced functionalities like timeslot allocation, overlap detection, and workload management. It modernizes traditional scheduling by automating timetable creation, ensuring a seamless academic environment.

Key Modules

Home Module

Central interface for managing academic schedules. Allows adding and managing classes, semesters, and predefined subjects.

Staff Management Module

Handles staff administration tasks, such as adding or removing staff and managing their schedules.

Edit Module

Enables selecting a class and adding/removing subjects. Also facilitates assigning and managing staff for specific subjects.

Schedule Module

Allocates class and lab schedules, ensuring no conflicts using the Overlap Detection System (ODS). Also manages tooltip alerts for overload warnings.

Export Module

Generates and exports timetables in user-friendly formats, including PDFs and Excel files for sharing among staff and students.

Tooltip Notification System

Provides real-time alerts for potential scheduling conflicts, such as overloaded staff or continuous teaching periods, to ensure balanced workloads.

Staff Details Module

Add or delete staff members from the system, and manage their schedules and assigned subjects.

Workload Module

Helps to distribute the workload evenly among staff, avoiding overload by monitoring the number of hours assigned to each faculty member.

Reset and Truncate Module

Offers options to reset or truncate all tables and unallocate all assigned staff, ensuring flexibility for major adjustments or new semesters.
